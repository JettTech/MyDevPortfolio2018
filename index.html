My Experience


While enrolled in the UT Austin Full Stack Bootcamp this past year, I paired with partners as well as led teams as project manager to accomplish our project goals. I am always learning and seeking out channels to develop test-driven applications and interactive User Interface (UI) web designs. Discovering the needs and methods to create a dynamic user experience (UX) in a secure way has made me a more knowledgeable programmer. Below are insights into the three main roles I've held while building my recent web applications.

Web Designer
Focusing on an animated, interactive, and responsive UI is key. Learning how to create eye-catching designs with an intuitive application flow is a very nuanced art, which I have discovered requires much research, field study, and mock user trials. During this process, my teams and I have made use of paper prototyping and image mirroring tools, like Figma.

Web Developer
Function, performance, and efficiency are key components when developing new products. Strong security is also a tenant of modern development as it protects application data and user identity and encourages customer loyalty. While learning to develop apps with each of these facets in mind, I became familiar with wireframing tools, prototyping styles and CoffeeScript test suites.

Project Manager
Nimbleness is key when creating innovative projects within strict deadlines. I strive to work alongside teammates with the Agile approach in mind. In past projects, my team and I sought out tools to streamline and clarify communication; we made use of task organizers and admin communication tools such as Github Project or Trello and always made sure to sort our backlog by priority.
